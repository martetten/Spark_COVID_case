# Разработка системы анализа медицинских изображений для эпидемиологического мониторинга COVID-19
Итоговый проект по дисциплине «Инфраструктура Big Data»

## 1. Цель и задачи

### Цель проекта
Разработать аналитическую систему для эпидемиологического мониторинга COVID-19 на основе метаданных рентгеновских снимков (репозиторий https://github.com/ieee8023/covid-chestxray-dataset/tree/master), используя стек PySpark.

### Задачи
Произвести первичную обработку данных, статистический анализ с помощью средств Spark/SQL, визуализировать результаты и оформить выводы.

## 4. Ноутбук
Все обработки осуществляются последовательным выполнением ячеек analysis.ipynb

Проект показал, что пик заболеваемости легочными болезнями в 21 веке пришелся на 2020 год и связан с пандемией COVID-19. Наибольшая частотность заболевания COVID-19 наблюдается у возрастной группы 56-70 лет. Чаще всего заболевают в январе. Распределение наблюдений по полу говорит о том, что данным болезням чаще подвержены мужчины. Проекция PA чаще всего используется в качестве первичного анализа и является наиболее часто используемой в рентгенографии проекцией. Для диагностики COVID-19 также необходимы снимки в проекциях AP и AP Supine.

Основные трудности в проведении анализа были связаны с неполнотой данных. Пропущенные значения пола и возраста заполнялись медианными значениями, характерными для диагноза конкретного наблюдения, исходя из принципа морбидоспецифичности. Для улучшения системы можно рекомендовать более строгую валидацию данных на этапе сбора.
